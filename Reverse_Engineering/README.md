# Reverse Engineering
---
This directory contains writeups for reverse engineering challenges solved during CTF competitions. These challenges focus on understanding how a program works internally and extracting hidden logic or secrets from compiled binaries.

The writeups here break down binaries step by step:

>Analyzing executable behavior
>Disassembling and decompiling code
>Identifying control flow and logic flaws
>Reconstructing algorithms
>Extracting hardcoded or computed flags
